Selamat Datang di Blog Pribadi Dira

Ini adalah halaman utama dari blog pribadi saya. Di blog pribadi ini saya menampilkan berita tentang pengalaman tanding saya beberapa tahun yang lalu.

Sebelumnya saya mau cerita sedikit, saya adalah anak tunggal, karena saya anak tunggal saya berusaha ingin membanggakan kedua orang tua saya, maka dari itu saya mengikuti kegiatan non akademik, karena saya merasa diri saya mampu untuk membanggakan kedua orang tua saya di prestasi saya di bidang Taekwondo ini. Saya merupakan atlet Taekwondo dari Kab. Sarolangun, dan penyandang sabuk Hitam DAN 1, saya mulai gabung taekwondo dari saya kelas 4 SD sampai sekarang, kurang lebih sudah 9 tahun saya gabung pada cabor taekwondo.

Saya sudah pernah mengikuti 9 event pertandingan, mulai dari antar Kabupaten, Provinsi, sampai Nasional. Di tahun 2021 saya mewakili provinsi jambi di event Bupati Lebong Cup dan mendapatkan medali Emas, lanjut di tahun 2022 saya mewakili provinsi jambi pada event kejuaraan Wilayah di Palembang dan mendapatkan medali Emas, di tahun 2023 saya kembali mewakili provinsi Jambi pada event kejuaraan Nasional Wilayah II di bengkulu mendapatkan medali Perunggu, di tahun yang sama saya mewakili kabupaten Sarolangun dalam event Porprov di event ini saya belum di kasih rezeki untuk berdiri di podium Porprov. Banyak lagi event yang belum saya sebutkan. Di masa sekarang saya kurang aktif lagi karena sudah sibuk dengan dunia perkuliahan, tapi tidak menutup kemungkinan saya untuk tidak bisa berprestasi lagi. Saya masih sering ikut acara acara taekwondo, tapi bukan sebagai atlet lagi melainkan sebagai wasit atau sebagai coach.
